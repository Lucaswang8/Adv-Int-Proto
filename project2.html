Project 2 : Spiral Pattern Generator

An interactive project where you use processing to generate spiral patterns.

Week 1 : Coding the Generator with one external control

The initial code without external controls

Inspiration

I saw a tutorial regarding generative design and they were making a spiral generator so I tried out the coding exercise. However, they were using mouseX as the way to interact with the spiral's form. It did not feel that intuitive for me hence I started to wonder if I could use the potentiometer and other controls to make a spiral pattern generator that I can reuse as a background for my visual designs.

Original Code

Version 1 : Replacing MouseX

I was deciding whether I should replace the mouseX or the "2500" variable. I initially started by replacing mouseX with the serialnumber coming from the Potentiometer

Version 2 : Replacing Variable

I found out that replacing mouseX still made the spiral move but it also reponded to the rotation interaction from the potentiometer. However, I wanted to control the motion entirely hence I replaced the 2500 variable in the map feature instead.
